Mockup d'une direction visuelle pour Kdai. L'idée : sortir du langage générique "rouge-orange-criard" des comparateurs concurrents (Honey, Rakuten, ShopBack) et viser un cran au-dessus. Ce qui change

Plus d'emoji géant écrasant au centre du hero. La typo (Instrument Serif pour les titres et les prix) porte l'identité. Le hero raconte une histoire en 3 lignes : produit, prix, action.

1 ligne = 1 décision

Les cartes "deal" sont compactes, comparables d'un coup d'œil. Pill colorée pour le type d'offre (1+1, −25 %, Stock bas), prix nouveau vs ancien à droite. Tap = fiche détail. Pas de second clic pour comparer.

Header & nav minimalistes

Plus de bandeau "📡 20 promos actives · 9 magasins" qui se voulait vivant mais bruyant. Le pulse-dot dans le hero (Deal du jour) communique la fraîcheur sans l'imposer. Bottom-nav 4 onglets, icônes 1.6 stroke, label en dessous.

Couleurs comme signal, pas comme décor

Le terracotta n'apparaît que sur les actions principales et les badges promo. Le sage = économies déjà acquises. Le plum = premium/spécial. L'amber = urgence (stock, dates). Chaque couleur dit quelque chose.

Si on valide cette direction

L'app entière (les 4 vues, modals, panneaux) sera refondue dans ce langage : cream + ink, Instrument Serif pour les prix et titres, Inter pour le corps, palette de pills sage / amber / plum / accent / ghost. Les emojis disparaissent des UI d'action, ne restent que sur les illustrations produit. Les SVG sont stroke 1.6 (plus fin qu'avant), sortis d'un set cohérent. Bottom nav restreinte sur mobile, sidebar + rail sur desktop. Pas de gradient agressif, pas de pulse infini, pas de "made with ❤️". L'app prend l'air d'un produit qu'on assume.
